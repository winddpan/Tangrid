## 1.4.1 beta
- 工作区现支持浏览器 URL、文档路径、Shell 命令和 AppleScript，一键还原工作现场。
- 提升工作区多窗口恢复速度。
- 优化休眠唤醒后的窗口位置恢复逻辑。

## 1.4.0 beta
- 修复 preferTabbed 相关问题
- 新增工作区（Workspace）功能

## 1.3.4 beta
- 修复了多个导致布局异常的问题
- 修复了一个快捷键异常的问题
- 修复旧版本macos窗口过滤设置没有显示保存按钮的问题

## 1.3.3 beta
- Dock预览新增媒体播放器
- 修复一个导致休眠恢复无法恢复布局的问题
- 修复了一个异常触觉反馈的bug

## 1.3.2 beta
- 优化窗口切换器的唤起速度。
- 修复多项布局问题，包括 Mission Control、`preferSplit` / `preferTabbed` 及全屏填充相关场景。

## 1.3.1 beta
- 修复一个特定情况下窗口管理状态丢失的问题
- 新增"显示器具有单独的空间"未开启的提示

## 1.3.0 beta
- 重写布局引擎，提升窗口管理稳定性。
- 修复 Tabbed 窗口在创建、切换或关闭标签页时可能导致的布局混乱。
- 修复设备从休眠恢复后可能出现的布局异常。
- 支持通过拖拽标签页来调整窗口布局。
- 优化 Dock 预览悬停体验，并改进三角区菜单悬停判定，减少误触。
- 修复标签栏闪烁和重复显示的问题。
- 支持按不同显示器分别设置间距和屏幕边缘。
- 重新设计 Auto Flow 模式下的窗口过渡动画。
